Just 3 Quick Steps

There is zero installation required. Just extract the ZIP archive, and you already up and running.

In a nutshell, install GRAV is a matter of unzipping the file to your web server.

Content is just simple markdown files in folders, with no complicated databases to deal with.

A powerful API and extensive hooks allows for plugins to extend Grav, such as the Admin Plugin.





From Simple to Sophisticated


Grav has the flexibility to power all sorts of websites. Flexible Twig templating allows easy realizations of your designs.
